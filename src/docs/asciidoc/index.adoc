== 개요

User Service는 회원가입·로그인·로그아웃·토큰 재발급 등 인증 기능과
사용자 정보 조회·수정·탈퇴·HOST 신청 기능을 제공합니다.
관리자(ADMIN)는 사용자 관리 및 HOST 신청 승인/거절을 수행합니다.

=== 인증 방식

모든 인증이 필요한 API는 API Gateway가 검증 후 주입하는 헤더를 사용합니다.

[cols="1,2,1"]
|===
| 헤더 | 설명 | 필수 여부

| `X-User-Id`
| 인증된 사용자의 Keycloak UUID
| 인증 필요 API

| `X-User-Role`
| 인증된 사용자의 역할 (CUSTOMER / HOST / ADMIN)
| 인증 필요 API
|===

=== 공통 응답 형식

성공·실패 모두 `ApiResponse<T>` 형식으로 응답합니다.

[source,json]
----
{
  "success": true,
  "code": "USER_FOUND",
  "message": "사용자 정보를 조회했습니다.",
  "timestamp": "2026-05-01T12:00:00",
  "data": { ... }
}
----
